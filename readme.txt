== Description ==
Tailored HTML 1.0 is a great way to dynamically control content for authenticated / unauthenticated users.

You can apply your configurations to pages/posts via ShortCode or Widgets. 

== Installation ==

<h4>If Tailored HTML was downloaded through WordPress:</h4>

<ol>
<li>Activate the plugin through the 'Plugins' menu in WordPress.</li>
<li>Begin configuring Tailored HTML by going to Settings -> Tailored HTML after the plugin has been installed.</li>
</ol>

<h4>If Tailored HTML was downloaded from <a href="https://www.datasprings.com">www.datasprings.com</a></h4>

<ol>
<li>After you have downloaded "ds-tailored-html.zip" from Data Springs go to your WordPress site(while logged in as an admin).</li>
<li>Go to Plugins -> Add New -> and click the "Upload Plugin" button at the top-left of this page.</li>
<li>You'll be taken to a page where you can upload the .zip that you downloaded. After uploading this .zip, click the "Install Now" button.</li>
<li>Tailored HTML should have been installed. If you had any issues, please contact us at <a href="mailTo:wordpress@datasprings.com">wordpress@datasprings.com</a>. Please note, it may take 1-2 business days to review and provide you a response.</li>
</ol>
